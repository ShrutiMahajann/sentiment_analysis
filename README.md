# 🤖 AI Sentiment Analyzer

AI Sentiment Analyzer is a modern web application built using **Flask** and **TextBlob** that analyzes the emotional tone of user-entered text. The application classifies text into **Positive**, **Negative**, or **Neutral** sentiment and presents the results through a professional, interactive dashboard with detailed analytics.

---

## 📌 Features

- 😊 Overall Sentiment Detection (Positive, Negative, Neutral)
- 📝 Sentence-wise Sentiment Analysis
- 📊 Sentiment Strength Meter
- 📈 Polarity Score Analysis
- 💭 Subjectivity Score Analysis
- 📄 Word Count
- 🔤 Character Count
- 📋 Total Sentence Count
- 💡 AI Insight based on text analysis
- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- ✨ Smooth Animations using AOS
- ⚡ Interactive Dashboard

## 📖 How It Works

1. Enter any text into the input box.
2. Click **Analyze Sentiment**.
3. The application processes the text using **TextBlob**.
4. It calculates:
   - Overall Sentiment
   - Polarity
   - Subjectivity
   - Word Count
   - Character Count
   - Sentence Count
5. Every sentence is analyzed individually.
6. Results are displayed in a modern analytics dashboard.
